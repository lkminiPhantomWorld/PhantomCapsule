## 這是什麼

幻影膠囊是一種可視化容器。

它不是單純圖片、普通 ZIP 或只有圖示的捷徑。

一顆幻影膠囊至少承載：Identity、Locator、Manifest、SHA256、Snapshot、ReverseChain、顯示面、回推鏈，以及啟動或導向入口。

## 公開範圍

本開源包只公開通用規格與可重用範本。
