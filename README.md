## 這是什麼

幻影膠囊是一種可視化容器。

它不是單純圖片、不是普通 ZIP、不是只有圖示的捷徑。

一顆幻影膠囊至少要能承載：身份、路徑、狀態、定位器、Manifest、SHA256、ReverseChain、顯示面、回推鏈、啟動或導向入口。

## 公開範圍

本開源包只公開通用規格與可重用範本。

不公開：🥃LKMINI 本體內容、🎩永恆核心私密條文、私密任務回執、未授權素材、私人 Drive / Goodnotes / iOS 捷徑內容、任何私人帳號、聯絡方式、內部路徑。

## A==A 原則

顯示面可以變。投影平台可以變。但身份鏈必須能回推同一來源。
